# AnalizadorDeFiniquitos
Analiza finiquitos para ver el comportamiento de lo ejecutado contra lo contratado y entrega una propuesta de inspección física

# Instrucciones
1. Carga tu archivo 1000_02
2. Señala cuantos renglones debe ignorar (por defecto son 11)
3. Señala el nombre de la hoja donde está el finiquito (hasta 2026 es la 12)
4. Indica el nombre de tu archivo incluyendo su extensión (.xlsx o .xlsm)
5. Clic en Analizar
